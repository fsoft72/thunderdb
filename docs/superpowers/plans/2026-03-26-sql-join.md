# SQL JOIN Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add INNER, LEFT, and RIGHT JOIN support to ThunderDB's SQL layer with multi-table chaining, table aliases, equality ON conditions, and dot-qualified column references.

**Architecture:** All JOIN logic lives in the REPL's `execute_select()`. The parser produces a new `FromClause` AST; when the REPL detects a join, it takes a separate execution path: flatten join chain, build merged column mapping, partition WHERE filters for pushdown, scan each table, hash-join sequentially, then apply post-join WHERE/ORDER BY/LIMIT/projection. The existing single-table query pipeline is untouched.

**Tech Stack:** Rust, ThunderDB parser/REPL/storage layer.

**Spec:** `docs/superpowers/specs/2026-03-26-sql-join-design.md`

```rust
    /// Parse FROM clause with optional JOIN chain
    ///
    /// Syntax: FROM table [alias] [JOIN table [alias] ON col = col] ...
    fn parse_from_clause(&mut self) -> Result<FromClause> {
        // Parse base table
        let name = self.expect_identifier()?;
        let alias = self.parse_optional_alias();
        let mut from = FromClause::Table(TableRef { name, alias });

        // Parse optional JOIN chain
        loop {
            let join_type = match self.current() {
                Token::Join => {
                    self.advance();
                    JoinType::Inner
                }
                Token::Inner => {
                    self.advance();
                    self.expect(Token::Join)?;
                    JoinType::Inner
                }
                Token::Left => {
                    self.advance();
                    let _ = self.match_token(&Token::Outer); // optional OUTER
                    self.expect(Token::Join)?;
                    JoinType::Left
                }
                Token::Right => {
                    self.advance();
                    let _ = self.match_token(&Token::Outer); // optional OUTER
                    self.expect(Token::Join)?;
                    JoinType::Right
                }
                _ => break,
            };

            // Parse right table
            let right_name = self.expect_identifier()?;
            let right_alias = self.parse_optional_alias();
            let right = TableRef { name: right_name, alias: right_alias };

            // Parse ON condition (equality only)
            self.expect(Token::On)?;
            let on_left = self.parse_column_ref()?;
            self.expect(Token::Equals)?;
            let on_right = self.parse_column_ref()?;

            from = FromClause::Join {
                left: Box::new(from),
                join_type,
                right,
                on_left,
                on_right,
            };
        }

        Ok(from)
    }

    /// Parse optional table alias (identifier that isn't a keyword)
    fn parse_optional_alias(&mut self) -> Option<String> {
        // An alias is an identifier that follows a table name, but isn't a keyword
        // like WHERE, JOIN, ON, ORDER, LIMIT, etc.
        match self.current() {
            Token::Identifier(name) => {
                let alias = name.clone();
                self.advance();
                Some(alias)
            }
            _ => None,
        }
    }

    /// Parse a column reference: identifier or identifier.identifier
    fn parse_column_ref(&mut self) -> Result<ColumnRef> {
        let first = self.expect_identifier()?;
        if self.match_token(&Token::Dot) {
            let column = self.expect_identifier()?;
            Ok(ColumnRef { table: Some(first), column })
        } else {
            Ok(ColumnRef { table: None, column: first })
        }
    }
```

```rust
use crate::parser::ast::{FromClause, JoinType, TableRef, ColumnRef};

/// Flattened join step extracted from the FromClause tree
struct JoinStep {
    join_type: JoinType,
    right: TableRef,
    on_left: ColumnRef,
    on_right: ColumnRef,
}

/// Flatten a FromClause join tree into a base table + ordered join steps
fn flatten_joins(from: &FromClause) -> (TableRef, Vec<JoinStep>) {
    match from {
        FromClause::Table(t) => (t.clone(), vec![]),
        FromClause::Join { left, join_type, right, on_left, on_right } => {
            let (base, mut steps) = flatten_joins(left);
            steps.push(JoinStep {
                join_type: join_type.clone(),
                right: right.clone(),
                on_left: on_left.clone(),
                on_right: on_right.clone(),
            });
            (base, steps)
        }
    }
}
```

- [ ] **Step 2: Add the merged column mapping builder**

```rust
/// Build a column mapping for joined tables.
///
/// Returns (qualified_map, column_count_per_table) where qualified_map
/// maps both "alias.col" and bare "col" (if unambiguous) to absolute
/// position in the merged row.
fn build_join_column_mapping(
    tables: &[(String, Option<String>, Vec<String>)],  // (name, alias, column_names)
) -> Result<HashMap<String, usize>> {
    let mut mapping = HashMap::new();
    let mut bare_count: HashMap<String, usize> = HashMap::new(); // count occurrences
    let mut bare_pos: HashMap<String, usize> = HashMap::new();   // last position
    let mut offset = 0usize;

    for (name, alias, columns) in tables {
        let qualifier = alias.as_deref().unwrap_or(name.as_str());
        for (i, col) in columns.iter().enumerate() {
            let pos = offset + i;
            // Always add qualified: "alias.column"
            mapping.insert(format!("{}.{}", qualifier, col), pos);
            // Track bare name occurrences
            *bare_count.entry(col.clone()).or_insert(0) += 1;
            bare_pos.insert(col.clone(), pos);
        }
        offset += columns.len();
    }

    // Add bare names only if unambiguous
    for (col, count) in &bare_count {
        if *count == 1 {
            if let Some(&pos) = bare_pos.get(col) {
                mapping.insert(col.clone(), pos);
            }
        }
    }

    Ok(mapping)
}
```

- [ ] **Step 3: Add the filter partitioning function**

```rust
/// Partition WHERE filters into per-table pushdowns and post-join filters.
///
/// A filter is pushable if its column reference resolves to exactly one table.
/// Pushdown filters use bare column names (for the single-table scan).
fn partition_filters(
    where_clause: &Option<crate::parser::ast::Expression>,
    tables: &[(String, Option<String>, Vec<String>)],
) -> Result<(HashMap<String, Vec<Filter>>, Vec<Filter>)> {
    let mut pushdowns: HashMap<String, Vec<Filter>> = HashMap::new();
    let mut post_join = Vec::new();

    let filters = if let Some(expr) = where_clause {
        Executor::expression_to_filters(expr)?
    } else {
        return Ok((pushdowns, post_join));
    };

    for filter in filters {
        // Check if column is qualified (contains a dot)
        if let Some(dot_pos) = filter.column.find('.') {
            let qualifier = &filter.column[..dot_pos];
            let bare_col = &filter.column[dot_pos + 1..];
            // Find which table this qualifier refers to
            let table_name = tables.iter()
                .find(|(name, alias, _)| {
                    alias.as_deref() == Some(qualifier) || name == qualifier
                })
                .map(|(name, _, _)| name.clone());

            if let Some(name) = table_name {
                // Push down with bare column name
                pushdowns.entry(name).or_default()
                    .push(Filter::new(bare_col.to_string(), filter.operator.clone()));
            } else {
                post_join.push(filter);
            }
        } else {
            // Bare column — check if it belongs to exactly one table
            let mut found_table = None;
            let mut ambiguous = false;
            for (name, _, columns) in tables {
                if columns.contains(&filter.column) {
                    if found_table.is_some() {
                        ambiguous = true;
                        break;
                    }
                    found_table = Some(name.clone());
                }
            }
            if ambiguous || found_table.is_none() {
                post_join.push(filter);
            } else {
                pushdowns.entry(found_table.unwrap()).or_default().push(filter);
            }
        }
    }

    Ok((pushdowns, post_join))
}
```

```rust
/// Perform a hash join between left and right row sets.
///
/// `left_col_idx` and `right_col_idx` are the positions of the ON columns
/// in their respective row value arrays.
/// `right_col_count` is needed for NULL padding in LEFT/RIGHT joins.
fn hash_join(
    left_rows: &[Row],
    right_rows: &[Row],
    left_col_idx: usize,
    right_col_idx: usize,
    join_type: &JoinType,
    left_col_count: usize,
    right_col_count: usize,
) -> Vec<Row> {
    use crate::storage::Value;

    // Build hash map on right table's ON column
    let mut right_map: HashMap<Value, Vec<usize>> = HashMap::new();
    for (i, row) in right_rows.iter().enumerate() {
        if let Some(val) = row.values.get(right_col_idx) {
            right_map.entry(val.clone()).or_default().push(i);
        }
    }

    let mut results = Vec::new();
    let mut right_matched = vec![false; right_rows.len()];

    for left_row in left_rows {
        let left_val = left_row.values.get(left_col_idx);
        let matches = left_val.and_then(|v| right_map.get(v));

        if let Some(indices) = matches {
            for &ri in indices {
                right_matched[ri] = true;
                let mut values = left_row.values.clone();
                values.extend(right_rows[ri].values.clone());
                results.push(Row { row_id: left_row.row_id, values });
            }
        } else if matches!(join_type, JoinType::Left) {
            // No match — emit left + NULLs
            let mut values = left_row.values.clone();
            values.extend(std::iter::repeat(Value::Null).take(right_col_count));
            results.push(Row { row_id: left_row.row_id, values });
        }
    }

    // RIGHT JOIN: emit unmatched right rows as NULLs + right
    if matches!(join_type, JoinType::Right) {
        for (i, matched) in right_matched.iter().enumerate() {
            if !matched {
                let mut values: Vec<Value> = std::iter::repeat(Value::Null).take(left_col_count).collect();
                values.extend(right_rows[i].values.clone());
                results.push(Row { row_id: right_rows[i].row_id, values });
            }
        }
    }

    results
}
```

```rust
    /// Execute a SELECT with JOIN
    fn execute_join(&mut self, select: &crate::parser::SelectStatement) -> Result<Vec<Row>> {
        let (base_table, join_steps) = flatten_joins(&select.from);

        // Collect table metadata: (name, alias, column_names)
        let mut table_info: Vec<(String, Option<String>, Vec<String>)> = Vec::new();

        // Helper to get column names for a table
        let get_columns = |db: &Database, name: &str| -> Vec<String> {
            if let Some(t) = db.get_table(name) {
                if let Some(schema) = t.schema() {
                    return schema.columns.iter().map(|c| c.name.clone()).collect();
                }
            }
            vec![]
        };

        // Base table
        let base_cols = get_columns(&self.database, &base_table.name);
        table_info.push((base_table.name.clone(), base_table.alias.clone(), base_cols));

        // Right tables from join steps
        for step in &join_steps {
            let cols = get_columns(&self.database, &step.right.name);
            table_info.push((step.right.name.clone(), step.right.alias.clone(), cols));
        }

        // Build merged column mapping
        let column_mapping = build_join_column_mapping(&table_info)?;

        // Partition WHERE filters
        let (pushdowns, post_join_filters) = partition_filters(
            &select.where_clause, &table_info,
        )?;

        // Scan base table with its pushdown filters
        let base_filters = pushdowns.get(&base_table.name).cloned().unwrap_or_default();
        let mut current_rows = self.database.scan(&base_table.name, base_filters)?;
        let mut current_col_count = table_info[0].2.len();

        // Execute each join step
        for (i, step) in join_steps.iter().enumerate() {
            let right_info = &table_info[i + 1];
            let right_filters = pushdowns.get(&step.right.name).cloned().unwrap_or_default();
            let right_rows = self.database.scan(&step.right.name, right_filters)?;
            let right_col_count = right_info.2.len();

            // Resolve ON column indices
            let left_qualifier = step.on_left.table.as_deref().unwrap_or("");
            let right_qualifier = step.on_right.table.as_deref().unwrap_or("");

            let left_key = if left_qualifier.is_empty() {
                step.on_left.column.clone()
            } else {
                format!("{}.{}", left_qualifier, step.on_left.column)
            };
            let right_key_bare = &step.on_right.column;

            let left_col_idx = column_mapping.get(&left_key)
                .copied()
                .ok_or_else(|| crate::error::Error::Query(
                    format!("Column '{}' not found in join", left_key)
                ))?;

            // Right column index is relative to the right table (not merged)
            let right_col_idx = right_info.2.iter()
                .position(|c| c == right_key_bare)
                .ok_or_else(|| crate::error::Error::Query(
                    format!("Column '{}' not found in table '{}'", right_key_bare, step.right.name)
                ))?;

            current_rows = hash_join(
                &current_rows, &right_rows,
                left_col_idx, right_col_idx,
                &step.join_type,
                current_col_count, right_col_count,
            );
            current_col_count += right_col_count;
        }

        // Apply post-join WHERE filters
        if !post_join_filters.is_empty() {
            current_rows.retain(|row| {
                crate::query::direct::apply_filters(row, &post_join_filters, &column_mapping)
            });
        }

        // Apply ORDER BY
        if let Some(ref order_by) = select.order_by {
            if let Some(&col_idx) = column_mapping.get(&order_by.column) {
                let desc = order_by.direction == crate::parser::ast::OrderDirection::Desc;
                current_rows.sort_by(|a, b| {
                    let va = a.values.get(col_idx);
                    let vb = b.values.get(col_idx);
                    let cmp = match (va, vb) {
                        (Some(a), Some(b)) => a.cmp(b),
                        (Some(_), None) => std::cmp::Ordering::Greater,
                        (None, Some(_)) => std::cmp::Ordering::Less,
                        (None, None) => std::cmp::Ordering::Equal,
                    };
                    if desc { cmp.reverse() } else { cmp }
                });
            }
        }

        // Apply LIMIT/OFFSET
        if let Some(offset) = select.offset {
            current_rows = current_rows.into_iter().skip(offset).collect();
        }
        if let Some(limit) = select.limit {
            current_rows.truncate(limit);
        }

        // Apply SELECT projection
        let select_cols: Vec<(String, usize)> = select.columns.iter().filter_map(|col| {
            match col {
                SelectColumn::Star => None, // keep all
                SelectColumn::Column(name) => {
                    column_mapping.get(name).map(|&idx| (name.clone(), idx))
                }
                SelectColumn::QualifiedColumn(table, col) => {
                    let key = format!("{}.{}", table, col);
                    column_mapping.get(&key).map(|&idx| (key, idx))
                }
                _ => None,
            }
        }).collect();

        if !select_cols.is_empty() && !select.is_select_star() {
            current_rows = current_rows.into_iter().map(|mut row| {
                row.values = select_cols.iter()
                    .filter_map(|(_, idx)| row.values.get(*idx).cloned())
                    .collect();
                row
            }).collect();
        }

        Ok(current_rows)
    }
```
